### 10:30 AM - 11:00 AM: The Machine Learning Workflow

Understanding the ML process:
1. **Problem Definition**: What are we trying to predict?
2. **Data Collection**: Gathering relevant data
3. **Data Cleaning**: Preparing data (80% of ML work!)
4. **Model Training**: Teaching the computer to recognize patterns
5. **Evaluation**: Testing if the model works
6. **Deployment**: Using the model in real applications

**Key Insight:** A Machine Learning Engineer spends 80% of their time on data preparation, not building models!

## Key Concepts Learned Today

- **Machine Learning**: Teaching computers to learn from data without explicit programming
- **Supervised Learning**: Learning from labeled examples (input → output)
- **Unsupervised Learning**: Finding patterns in unlabeled data
- **Training Data**: Examples used to teach the model
- **Prediction**: Using a trained model to make decisions on new data
- **Feature**: An input variable used for prediction (e.g., rainfall, temperature)
- **Model**: The "brain" that learns patterns from data
