# Week 02: Machine Learning, Mobile Development & Kigali Web Artisans Projects

## Overview
Welcome to **Week 2** of your software engineering internship! This week introduces you to the exciting worlds of **machine learning** and **mobile development**, while exploring real-world projects built and maintained by **Kigali Web Artisans**. You'll gain exposure to modern technologies and see how professional software engineering teams build and maintain production applications.

This week bridges foundational skills from Week 1 with the web development focus in Week 3 and culminates in understanding the technologies you'll use in your final machine learning project in Week 4.

By the end of this week, you will:

- ✅ Understand the software development lifecycle in production environments
