# Week 01: Fundamentals & Development Environment Setup

## Overview
Welcome to **Week 1** of your software engineering internship! This week lays the foundation for your entire 20-day journey. You'll establish critical development skills, learn version control, master core programming languages, and build the technical infrastructure you'll use throughout the program.

This week is **essential preparation** for the advanced topics in Weeks 2-4, particularly your final machine learning project. Every skill you learn this week will be directly applied in your upcoming work.

---

## 🎯 Week Goals

By the end of this week, you will:
- ✅ Master Git and GitHub workflows for remote collaboration
- ✅ Set up a professional development environment
- ✅ Build practical skills in PHP, databases, and Node.js
- ✅ Develop problem-solving abilities through algorithmic challenges
- ✅ Learn code documentation and professional communication
- ✅ Understand the full-stack development pipeline

## 🧩 Weekly LeetCode Challenge

**Problem (Easy):** [Two Sum](https://leetcode.com/problems/two-sum/)

> Given an array of integers `nums` and an integer `target`, return indices of the two numbers such that they add up to `target`.

This problem reinforces array traversal and hash map usage — core concepts for backend data handling.
