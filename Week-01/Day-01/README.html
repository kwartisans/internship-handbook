Day 01 - Welcome to the Internship Program

Welcome Message from Engineering Manager

Welcome to our software engineering internship program! We are thrilled to have you join our team for this exciting 20-day journey. This program is designed to give you hands-on experience with real-world development practices and tools that professional engineers use every day.

As this is a remote internship, strong version control skills and self-discipline are essential. We've structured today to help you build a solid foundation. Remember, every expert was once a beginner—don't hesitate to ask questions and reach out to your mentors whenever you need support.

Looking forward to seeing what you'll build!

Best regards,
Engineering Manager

Overview

Welcome to Day 1 of your 20-day high school remote internship program! Today marks the beginning of your journey into software engineering.

Goals for Today

  • Get familiar with the internship structure
  • Complete prerequisite training on Git and GitHub
  • Set up your development environment
  • Join the community channels
  • Meet your team and mentors
  • Understand the program expectations

Schedule

8:00 AM - 9:00 AM: Welcome & Orientation

  • Review program overview and expectations
  • Introduction to mentors and team members

9:00 AM - 10:30 AM: Prerequisite Training - Version Control

As our internship is remote-based, it requires intensive knowledge of version control. To help you feel confident before you start, please complete the following prerequisite training for GitHub and Git:

  • GitHub Overview: Watch Video (15 minutes)
  • GitHub Tutorial: Watch Video (30 minutes)
  • Git Tutorial: Watch Video (30 minutes)

10:30 AM - 11:30 AM: Development Environment Setup

  • Install necessary tools and software
  • Configure your development environment
  • Test your setup with a simple project

11:30 AM - 12:30 PM: Hands-On Practice

  • Create your first GitHub repository
  • Practice basic Git commands
  • Make your first commit and push

12:30 PM - 1:00 PM: Community Onboarding

  • Join our Discord community: https://discord.gg/rrj8WxYp
  • Introduce yourself in the team channels
  • Get familiar with communication protocols

1:00 PM - 2:00 PM: Lunch Break

Tasks to Complete Today

✅ Required Tasks

  1. Join Discord Community: https://discord.gg/rrj8WxYp
  2. Complete Prerequisite Training:
    • GitHub Overview video
    • GitHub Tutorial video
    • Git Tutorial video
  3. Development Environment Setup:
    • Install Git
    • Install a code editor (VS Code recommended)
    • Configure Git with your name and email
  4. GitHub Practice:
    • Create your first GitHub repository
    • Make at least 3 commits
    • Practice pushing and pulling changes
  5. Introduce Yourself:
    • Post an introduction in the Discord channel
    • Share your background and what you hope to learn
