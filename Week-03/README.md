# Week 03: Web Development Fundamentals & Vibe Coding

## Overview
Welcome to **Week 3** of your software engineering internship! This week bridges your foundation from Weeks 1-2 with the final project in Week 4. You'll dive into **web development fundamentals**, learn **modern coding practices**, and explore **vibe coding** - an intuitive, creative approach to building interactive applications.

This is an **8-hour daily intensive** designed to prepare you for real-world web development and your final ML project.

---

## 🎯 Week Goals

By the end of this week, you will:
- ✅ Build responsive web pages with HTML and CSS
- ✅ Add interactivity using JavaScript
- ✅ Understand DOM manipulation and event handling
- ✅ Practice "vibe coding" - building projects through experimentation and creativity
- ✅ Use Git workflows for collaborative development
- ✅ Complete a mini web application project

---

## 📚 Essential Learning Philosophy

**Vibe Coding**: This week introduces you to a creative, flow-state approach to coding where you:
- Build by doing, not just by following tutorials
- Experiment and iterate rapidly
- Focus on the end user experience
- Learn through curiosity and exploration
- Make mistakes and learn from them quickly

## 🧩 Weekly LeetCode Challenge

**Problem (Easy):** [Reverse String](https://leetcode.com/problems/reverse-string/)

> Write a function that reverses a string in place. The input is given as an array of characters `s`.

This problem reinforces in-place array manipulation and two-pointer technique — useful when building interactive JavaScript features.
