# Day 15 Portfolio Starter

## What's Included
- `index.html` - Basic HTML structure
- `styles.css` - Starter CSS with responsive design
- `script.js` - JavaScript for interactivity

## Getting Started

1. Customize the HTML with your information
2. Style it to match your personality
3. Add JavaScript for interactivity
4. Test on different devices
5. Deploy to GitHub Pages
