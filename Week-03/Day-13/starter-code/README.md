# Day 13 Vibe Coding Starter

## Philosophy
Vibe coding is about creative flow, experimentation, and building what feels right!

## Quick Start Projects
Choose one or build all three:
1. Color Palette Generator
2. Quote Generator
3. Calculator

## How to Vibe Code
1. **Start with an idea** - even a vague one
2. **Build rapidly** - don't overthink
3. **Experiment freely** - try different approaches
4. **Iterate quickly** - make it better with each pass
5. **Ship it** - done is better than perfect

## Starter File
- `vibe-starter.html` - Basic template to start from

## Tips
- Use your browser console (F12) for debugging
- Don't be afraid to break things
- Copy concepts, not code
- Make it yours
