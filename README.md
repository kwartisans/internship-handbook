# internship-handbook
This is a handbook for our 4-week software engineering internship program for high school students.

## Program Structure

This 20-day remote internship program is organized into 4 weeks, with 5 days per week:

- **Week 01**: Days 01-05
- **Week 02**: Days 06-10
- **Week 03**: Days 11-15
- **Week 04**: Days 16-20

Each day has its own folder with a README.md containing the day's schedule, goals, resources, and homework.

## Setup Script

We've included a Bash script to set up the directory structure. If you need to recreate or set up the structure elsewhere, you can use:

```bash
bash setup-internship-structure.sh
```

The script will:
1. Create a main `internship-handbook` directory (or use the current directory if already inside it)
2. Create Week-01 through Week-04 folders
3. Create Day folders with sequential numbering (Day-01 to Day-20)
4. Add a README.md file to each day folder

## Getting Started

Navigate to [Week-01/Day-01](Week-01/Day-01/README.md) to begin your internship journey!
